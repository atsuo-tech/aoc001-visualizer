Tools 使用方法

Tools をダウンロード (Windows版)し、コマンドプロンプトを利用して操作してください。
Windows 版以外には、tools (Ubuntu版)があります。
どちらにも該当しない場合は、ビルド前のtools.rsをビルドして利用してください。

テストケース生成

一つだけ

tools gen once ./input 1234

ランダムシード 1234 を用いてテストケースを一つ生成し、input フォルダに保存します。
ファイル名は input_1234.txt となります。

ファイルから

tools gen list ./input seed.txt

seed.txt に記載されたランダムシードを用いて複数のテストケースを生成し、input フォルダに保存します。
ファイル名は input_<シード値>.txt となります。

ランダムに

tools gen random ./input 2000

ランダムに 2000 個のテストケースを生成し、input フォルダに保存します。
ファイル名は input_<シード値>.txt となります。

テスト

tools test dir ./input ./Main.exe

input フォルダ内のすべてのテストケースを用いて Main.exe をテストします。
